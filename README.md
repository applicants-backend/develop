
<h1>1.프로젝트명</h1>
운동입문자를 위한 정보 및 운동기록 웹사이트

<h1>2.프로젝트 설명</h1>
프로젝트 내용 : 새로운 운동을 입문하고 싶은 사람들이 가볍게 운동에 대한 정보를 접하고 기록할 수 있는 페이지를 만들었다.
<br/>
프로젝트 목적 : js, jquery를 활용하여 동적인 페이지를 제작하고 반응형으로 모두 처리하기
<h1>3.개발 환경 </h1>
: VSC
<h1>4.사용 기술 :</h1>

css, javascript , jquery
